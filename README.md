# PetalFloat
PetalFloat是一个Web前端花瓣飘落效果插件
### 使用方法
* 引入petalfloat.js
```
<script src="js/petalfloat.js" type="text/javascript"></script>
```
* 添加如下代码
```
<script type="text/javascript">
  var pf = new PF();
  pf.init();
  pf.start();
</script>
```
